# `if __name__ == '__main__':` 是什么意思？

## 问题

```python
if __name__ == '__main__':
```

这行代码是什么意思？为什么这样写？

## 回答

**`__name__` 是 Python 自动赋值的内置变量：**

| 运行方式 | `__name__` 值 | 结果 |
|---------|--------------|------|
| `python app.py` | `'__main__'` | ✓ 执行代码 |
| `from app import xxx` | `'app'` | ✗ 跳过，不执行 |

**为什么需要？**

1. **避免重复执行** - 当其他模块导入 app.py 时，不会启动服务器
2. **明确入口点** - 只在直接运行时才启动 uvicorn
