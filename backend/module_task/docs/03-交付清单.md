## 3. 数据库表设计

### 3.1 biz_daily_task
- 支持 3 种任务类型：daily（每日）、once（一次性）、long（长期）
- 支持 3 种状态：pending（待完成）、completed（已完成）、disabled（已禁用）
- 支持置顶、排序、完成次数统计
- 自动记录最后完成时间和禁用时间

### 3.2 biz_daily_task_category
- 支持自定义分类
- 支持排序和图标

### 3.3 biz_daily_task_log
- 记录任务操作历史
- 支持审计追溯

## 6. API 使用示例

### 6.1 创建每日任务

```http
POST /daily-task HTTP/1.1
Content-Type: application/json
Authorization: Bearer <token>

{
  "title": "早起打卡",
  "description": "每天早上 6 点起床",
  "taskType": "daily",
  "iconType": "calendar"
}
```

### 6.2 完成任务

```http
PATCH /daily-task/1/complete HTTP/1.1
Authorization: Bearer <token>
```

### 6.3 批量排序

```http
PUT /daily-task/reorder HTTP/1.1
Content-Type: application/json
Authorization: Bearer <token>

{
  "taskOrders": [
    {"taskId": 1, "sortOrder": 1},
    {"taskId": 2, "sortOrder": 2}
  ]
}
```
