## 每日任务管理接口

### 1. 获取任务列表

#### 1.1 基本信息
- **接口说明**：根据条件查询每日任务列表，支持分页
- **请求方式**：GET
- **接口路径**：`/api/daily-task/list`
- **权限标识**：`daily:task:list`

#### 1.2 请求参数

**查询参数：**

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| title | string | 否 | - | 任务标题（模糊查询） |
| taskType | string | 否 | - | 任务类型（daily/once/long） |
| status | string | 否 | - | 状态（pending/completed/disabled） |
| isPinned | boolean | 否 | - | 是否置顶 |
| categoryId | int | 否 | - | 分类ID |
| beginTime | string | 否 | - | 开始时间（格式：yyyy-MM-dd） |
| endTime | string | 否 | - | 结束时间（格式：yyyy-MM-dd） |
| pageNum | int | 否 | 1 | 页码 |
| pageSize | int | 否 | 10 | 每页数量 |

**taskType枚举值：**
| 值 | 说明 |
|----|------|
| daily | 每日任务 |
| once | 一次性任务 |
| long | 长期任务 |

**status枚举值：**
| 值 | 说明 |
|----|------|
| pending | 待完成 |
| completed | 已完成 |
| disabled | 已禁用 |

```
GET /api/daily-task/list?status=pending&pageNum=1&pageSize=10
```
