## 五、阴影系统

### 5.1 阴影层级

| 层级 | CSS | 用途 |
|------|-----|------|
| sm | `0 1px 2px rgba(0,0,0,0.05)` | 轻微抬起 |

```css

--shadow-md: 0 4px 6px rgba(0, 0, 0, 0.3);
```

## 八、动画与过渡

### 8.1 过渡时长

| 变量 | 值 | 用途 |
|------|-----|------|
| `--duration-fast` | 150ms | 按钮悬停 |
| `--duration-base` | 200ms | 颜色变化 |
| `--duration-slow` | 300ms | 展开/收起 |
| `--duration-slower` | 500ms | 页面切换 |

### 8.2 缓动函数

| 变量 | 值 | 用途 |
|------|-----|------|
| `--ease-out` | cubic-bezier(0, 0, 0.2, 1) | 标准过渡 |
| `--ease-in` | cubic-bezier(0.4, 0, 1, 1) | 进入动画 |
| `--ease-in-out` | cubic-bezier(0.4, 0, 0.2, 1) | 往返动画 |

### 8.3 关键动画

#### 打字机效果
```css
@keyframes blink {
  0%, 50% { opacity: 1; }
  51%, 100% { opacity: 0; }
}

.cursor {
  display: inline-block;
  width: 2px;
  height: 16px;
  background: currentColor;
  animation: blink 1s infinite;
}
```

#### 思考过程脉冲
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}

.thinking-indicator {
  animation: pulse 1.5s infinite;
}
```

#### 消息淡入
```css
@keyframes fade-in {
  from {
    opacity: 0;
    transform: translateY(8px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.message-item {
  animation: fade-in 300ms ease-out;
}
```

## 十一、可访问性

### 11.1 焦点管理

| 元素 | 焦点样式 |
|------|----------|
| 可聚焦元素 | 2px 主色轮廓, offset 2px |
| 跳过导航 | 首次加载聚焦到输入框 |
| 模态框 | 打开后聚焦到关闭按钮 |
| 错误提示 | 屏幕阅读器朗读 |

### 11.2 ARIA 属性

| 场景 | 属性 |
|------|------|
| 加载状态 | `aria-busy="true"` |
| 展开面板 | `aria-expanded="true/false"` |
| 消息角色 | `role="log" aria-live="polite"` |
| 输入框 | `aria-label="发送消息"` |
