# 聊天模块 - UI细节设计

> **目标**: 定义视觉设计系统、CSS变量、组件样式规范
> **原则**: 语义化、可维护、支持主题切换

## 🎨 颜色系统

### CSS 变量定义

```css
/* ===== 主题色板 ===== */
:root {
  /* 主色 */
  --primary-color: #4F46E5;
  --primary-light: #E0E7FF;
  --primary-dark: #3730A3;

  /* 中性色 */
  --gray-50: #F9FAFB;
  --gray-100: #F3F4F6;
  --gray-200: #E5E7EB;
  --gray-300: #D1D5DB;
  --gray-400: #9CA3AF;
  --gray-500: #6B7280;
  --gray-600: #4B5563;
  --gray-700: #374151;
  --gray-800: #1F2937;
  --gray-900: #111827;

  /* 语义色 */
  --success-color: #10B981;
  --warning-color: #F59E0B;
  --error-color: #EF4444;
  --info-color: #3B82F6;
}

/* 浅色主题 */
:root, [data-theme="light"] {
  --bg-primary: #FFFFFF;
  --bg-secondary: #F9FAFB;
  --bg-tertiary: #F3F4F6;

  --text-primary: #111827;
  --text-secondary: #6B7280;
  --text-disabled: #9CA3AF;

  --border-color: #E5E7EB;
}

/* 深色主题 */
[data-theme="dark"] {
  --bg-primary: #111827;
  --bg-secondary: #1F2937;
  --bg-tertiary: #374151;

  --text-primary: #F9FAFB;
  --text-secondary: #9CA3AF;
  --text-disabled: #6B7280;

  --border-color: #374151;
}
```

### 颜色使用约束

| 用途 | 浅色主题 | 深色主题 | CSS变量 |
|------|----------|----------|---------|
| 主按钮 | `#4F46E5` | `#4F46E5` | `--primary-color` |
| 主按钮悬停 | `#3730A3` | `#4338CA` | `--primary-dark` |
| 用户消息气泡 | `#4F46E5` | `#4F46E5` | `--primary-color` |
| AI消息气泡 | `#F3F4F6` | `#374151` | `--bg-tertiary` |
| 输入框边框 | `#E5E7EB` | `#374151` | `--border-color` |
| 错误提示 | `#EF4444` | `#EF4444` | `--error-color` |

### 消息气泡颜色

| 消息类型 | 浅色主题背景 | 浅色主题文字 | 深色主题背景 | 深色主题文字 |

## 🎬 动画与过渡

### 过渡时长

```css
:root {
  --duration-fast: 150ms;      /* 按钮悬停 */
  --duration-base: 200ms;      /* 颜色变化 */
  --duration-slow: 300ms;      /* 展开/收起 */
  --duration-slower: 500ms;    /* 页面切换 */
}
```

### 缓动函数

```css
:root {
  --ease-out: cubic-bezier(0, 0, 0.2, 1);      /* 标准过渡 */
  --ease-in: cubic-bezier(0.4, 0, 1, 1);       /* 进入动画 */
  --ease-in-out: cubic-bezier(0.4, 0, 0.2, 1); /* 往返动画 */
}
```

### 关键动画定义

#### 打字机光标效果

```css
@keyframes blink {
  0%, 50% { opacity: 1; }
  51%, 100% { opacity: 0; }
}

.typing-cursor {
  display: inline-block;
  width: 2px;
  height: 16px;
  background: currentColor;
  animation: blink 1s infinite;
}
```

#### 思考过程脉冲

```css
@keyframes pulse {
```


## ♿ 可访问性

### 焦点管理

```css
/* 可聚焦元素焦点样式 */
:focus-visible {
  outline: 2px solid var(--primary-color);
  outline-offset: 2px;
}

/* 移除默认焦点样式（仅保留键盘焦点） */
:focus:not(:focus-visible) {
  outline: none;
}
```

| 场景 | 属性 | 值 |
