# 聊天模块 - 路由设计

## 一、路由结构

### 1.1 整体路由树

```
/
├── /login                    # 登录页
├── /chat                     # 聊天主页
│   ├── /chat                 # 默认会话 (新建或最近)
│   ├── /chat/:conversationId # 指定会话
│   └── /chat/new             # 新建会话
├── /settings                 # 设置页
├── /404                      # 404 页面
└── /*                        # 重定向到 /chat
```

**路由参数**:

| 参数 | 类型 | 说明 | 默认值 |

## 三、路由配置代码

### 3.1 Vue Router 配置

```typescript
// router/index.ts
import { createRouter, createWebHistory, RouteRecordRaw } from 'vue-router'

const routes: RouteRecordRaw[] = [
  {
    path: '/login',
    name: 'login',
    component: () => import('@/pages/LoginPage.vue'),
    meta: {
      title: '登录',
      requiresAuth: false
    }
  },
  {
    path: '/chat',
    name: 'chat',
    component: () => import('@/pages/ChatPage.vue'),
    meta: {
      title: 'DeepSeek Chat',
      requiresAuth: true
    },
    children: [
      {
        path: ':conversationId',
        name: 'conversation',
        component: () => import('@/pages/ChatPage.vue'),
        props: true
      }
    ]
  },
  {
    path: '/settings',
    name: 'settings',
    component: () => import('@/pages/SettingsPage.vue'),
    meta: {
      title: '设置',
      requiresAuth: true
    }
  },
  {
    path: '/404',
    name: '404',
    component: () => import('@/pages/NotFoundPage.vue'),
    meta: {
      title: '页面未找到'
    }
  },
  {
    path: '/:pathMatch(.*)*',
    redirect: '/chat'
  }
]

const router = createRouter({
  history: createWebHistory(),
  routes,
  scrollBehavior(to, from, savedPosition) {
    if (savedPosition) {
      return savedPosition
    }
    // 聊天页面保持滚动位置
    if (to.name === 'conversation') {
      return false
    }
    return { top: 0 }
  }
})
```

## 四、路由守卫

### 4.1 全局前置守卫

```typescript
router.beforeEach(async (to, from, next) => {
  // 设置页面标题
  if (to.meta.title) {
    document.title = to.meta.title as string
  }

  // 检查登录状态
  const userStore = useUserStore()
  const requiresAuth = to.meta.requiresAuth !== false

  if (requiresAuth && !userStore.isAuthenticated) {
    // 未登录，跳转到登录页
    next({ name: 'login', query: { redirect: to.fullPath } })
    return
  }

  // 已登录访问登录页，跳转到聊天页
  if (to.name === 'login' && userStore.isAuthenticated) {
    next({ name: 'chat' })
    return
  }

  next()
})
```

### 4.2 路由独享守卫

```typescript
{
  path: '/chat/:conversationId',
  beforeEnter: (to, from, next) => {
    const conversationId = parseInt(to.params.conversationId as string)
    const conversationStore = useConversationStore()

    // 检查会话是否存在
    const conversation = conversationStore.conversations.find(
      c => c.conversationId === conversationId
    )

    if (!conversation) {
      // 会话不存在，可以跳转到 404 或重定向到聊天主页
      next({ name: 'chat' })
      return
    }

    next()
  }
}
```

### 5.1 路由参数传递

| 方式 | 示例 | 用途 |
|------|------|------|
| 路径参数 | `/chat/123` | 会话 ID |

### 5.2 参数获取

```typescript
// 路径参数
const route = useRoute()
const conversationId = computed(() => parseInt(route.params.conversationId as string))

// 查询参数
const modelId = computed(() => route.query.model as string)

// 监听路由变化
watch(
  () => route.params.conversationId,
  (newId) => {
    if (newId) {
      // 切换会话
      conversationStore.setCurrentConversation(parseInt(newId))
    }
  },
  { immediate: true }
)
```

## 六、路由导航

### 6.1 声明式导航

```vue
<!-- 导航到指定会话 -->
<router-link :to="{ name: 'conversation', params: { conversationId: 123 } }">
  Python 学习
</router-link>

<!-- 导航到新建会话 -->
<router-link :to="{ name: 'chat', query: { new: true } }">
  新建对话
</router-link>
```

### 6.2 编程式导航

```typescript
import { useRouter } from 'vue-router'

const router = useRouter()

// 导航到指定会话
router.push({ name: 'conversation', params: { conversationId: 123 } })

// 替换当前路由 (不产生历史记录)
router.replace({ name: 'conversation', params: { conversationId: 123 } })

// 返回上一页
router.back()

// 导航到新建会话
router.push({ name: 'chat' })
```

## 七、特殊场景处理

### 7.1 新建会话

**方案 1: 使用 query 参数**
```typescript
// 导航
router.push({ name: 'chat', query: { new: 'true' } })

// 组件内判断
const isNewConversation = computed(() => route.query.new === 'true')
```

**方案 2: 使用特殊路径**
```typescript
// 路由配置
{ path: 'new', name: 'new-conversation', component: ChatPage }

// 导航
router.push({ name: 'new-conversation' })
```

### 7.2 会话切换

```typescript
// 切换会话时保留滚动位置
const switchConversation = async (conversationId: number) => {
  const fromId = route.params.conversationId
  const toId = conversationId

  // 保存当前滚动位置
  if (fromId) {
    scrollPositions.value[fromId] = window.scrollY
  }

  // 导航到新会话
  await router.push({
    name: 'conversation',
    params: { conversationId: toId }
  })

  // 恢复滚动位置
  const savedPosition = scrollPositions.value[toId]
  if (savedPosition) {
    window.scrollTo(0, savedPosition)
  }
}
```

### 7.3 未保存提示

```typescript
// 在路由离开时检查是否有未发送的草稿
onBeforeRouteLeave((to, from, next) => {
  if (hasUnsavedContent.value) {

      next()
    } else {
      next(false)

    next()
```

## 八、路由与状态同步

### 8.1 路由 -> 状态

```typescript
// 监听路由变化，更新状态
watch(
  () => route.params.conversationId,
  async (newId, oldId) => {
    if (newId && newId !== oldId) {
      // 加载会话数据
      await conversationStore.setCurrentConversation(parseInt(newId))

      // 加载消息列表
      await messageStore.fetchMessages(parseInt(newId))
    }
  }
)
```

### 8.2 状态 -> 路由

```typescript
// 发送消息后更新 URL
const sendMessage = async (content: string) => {
  const userMessageId = await messageStore.sendMessage(
    conversationStore.currentConversationId,
    content
  )

  // 如果是新建会话，更新 URL
  if (route.name === 'chat' && !route.params.conversationId) {
    router.replace({
      name: 'conversation',
      params: { conversationId: conversationStore.currentConversationId }
    })
  }
}

// 创建新会话后导航
const createConversation = async () => {
  const conversation = await conversationStore.createConversation({
    title: '新对话',
    modelId: 'deepseek-chat'
  })

  router.push({
    name: 'conversation',
    params: { conversationId: conversation.conversationId }
  })
}
```

## 九、移动端路由处理

### 9.1 抽屉式导航

```typescript
// 移动端会话列表处理
const isMobile = computed(() => window.innerWidth < 768)

// 进入会话时隐藏侧边栏
watch(
  () => route.params.conversationId,
  (conversationId) => {
    if (isMobile.value && conversationId) {
      appStore.sidebarVisible = false
    }
  }
)
```

### 9.2 返回手势

```typescript
// 移动端返回按钮逻辑
const goBack = () => {
  if (isMobile.value) {
    if (appStore.sidebarVisible) {
      // 侧边栏打开，关闭侧边栏
      appStore.sidebarVisible = false
    } else if (route.params.conversationId) {
      // 在会话中，返回会话列表
      router.push({ name: 'chat' })
    } else {
      // 在会话列表，退出应用或提示
      if (window.confirm('确定要退出吗？')) {
        router.push({ name: 'login' })
      }
    }
  } else {
    router.back()
  }
}
```
