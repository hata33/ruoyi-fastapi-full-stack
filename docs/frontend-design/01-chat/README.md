## 📋 术语表

### 核心概念

| 中文术语 | 英文术语 | 定义 | API对应 |
|----------|----------|------|---------|
| 会话 | Conversation | 一组相关的消息对话，对应一个聊天窗口 | `chat_conversation` |
| 消息 | Message | 单次用户输入或AI回复 | `chat_message` |
| 对话 | Dialogue | 会话的别名，与Conversation互换使用 | - |
| 聊天区域 | ChatArea | 消息列表与输入框组成的交互区域 | - |
| 侧边栏 | Sidebar | 显示会话列表的左侧面板 | - |
| 流式响应 | Streaming / SSE | Server-Sent Events实现的实时输出 | `/api/chat/messages/stream` |
| 推理过程 | Thinking Process | Reasoner模型的深度思考内容 | `thinking_content` |
| 模型 | Model | AI对话模型（如deepseek-chat） | `chat_model` |

### UI组件术语

| 中文术语 | 英文术语 | 说明 |
|----------|----------|------|
| 消息气泡 | MessageBubble | 包含消息内容的圆角矩形 |
| 打字机效果 | Typewriter Effect | 文字逐字显示的动画 |
| 思考面板 | ThinkingPanel | 折叠式展示推理过程的组件 |
| 占位消息 | Placeholder Message | AI生成中的临时消息 |
| 骨架屏 | Skeleton | 加载时的占位动画 |

### 状态术语

| 中文术语 | 英文术语 | 说明 |
|----------|----------|------|
| 空闲 | Idle | 无消息正在生成的状态 |
| 生成中 | Generating | AI正在输出内容 |
| 思考中 | Thinking | AI正在进行深度推理 |
| 已停止 | Stopped | 用户中断生成 |
| 已完成 | Completed | 消息生成完成 |

## 🎯 文档约定

### 组件规格模板

每个组件遵循以下约束表格格式：

| 约束项 | 说明 | 必填 |
|--------|------|------|
| Props | 组件输入属性 | ✅ |
| Emits | 组件输出事件 | ✅ |
| Slots | 插槽定义 | ❌ |
| Ref Expose | 暴露的方法/属性 | ❌ |
| Dependencies | 依赖的其他组件 | ❌ |

### 代码示例约定

- **精简原则**: 只展示核心逻辑，省略错误处理、样式定义等
- **类型优先**: 使用 TypeScript 接口定义数据结构
- **单一职责**: 每个示例只演示一个功能点

### Mermaid 图约定

- **状态图**: 使用 `stateDiagram-v2` 描述状态流转
- **序列图**: 使用 `sequenceDiagram` 描述时序交互
- **流程图**: 使用 `graph` 描述决策流程
