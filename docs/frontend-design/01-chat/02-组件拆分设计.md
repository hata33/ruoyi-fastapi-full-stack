# 聊天模块 - 组件拆分设计

## 一、组件拆分原则

### 1.1 拆分原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 单一职责 | 一个组件只做一件事 | MessageList 只负责消息展示 |
| 可复用性 | 可在多处复用的内容独立为组件 | Avatar、Button |
| 独立状态 | 有独立交互逻辑的内容独立 | MessageItem (复制、删除) |
| 层级合理 | 不超过 4-5 层嵌套 | ChatArea > MessageList > MessageItem |
| 按需加载 | 非首屏内容拆分为独立组件 | ExportModal, TagManager |

### 1.2 组件分类

```
┌─────────────────────────────────────────────────────────────┐
│                      页面级组件 (Pages)                       │
│  ChatPage (唯一页面，使用动态路由)                            │
├─────────────────────────────────────────────────────────────┤
│                     布局组件 (Layouts)                       │
│  AppLayout, ChatLayout                                       │
├─────────────────────────────────────────────────────────────┤
│                    业务组件 (Features)                       │
│  Header, Sidebar, ChatArea, MessageList, InputArea...       │
├─────────────────────────────────────────────────────────────┤
│                    基础组件 (Common)                         │
│  Button, Input, Modal, Avatar, Tag, Badge...                │
└─────────────────────────────────────────────────────────────┘
```

## 二、页面级组件

### 2.1 ChatPage

**职责**: 聊天页面入口，负责路由和整体布局

**需要管理的状态**:
- 当前会话 ID
- 加载状态
- 错误状态

**不需要管理**:
- 会话列表数据 (Sidebar 管理)
- 消息列表数据 (ChatArea 管理)
- 模型配置 (Header 管理)

---

## 三、布局组件

### 3.1 AppLayout

**职责**: 应用整体布局框架

**包含内容**:
- Header (固定顶部)
- 主体内容区域
- 全局 Modal/Toast 容器

**不包含**: 业务逻辑

### 3.2 ChatLayout

**职责**: 聊天页面专用布局

**包含内容**:
- Sidebar (侧边栏)
- ChatArea (聊天主区域)
- 响应式切换逻辑

---

## 四、业务组件拆分

### 4.1 Header 组件

**拆分内容**:

| 子组件 | 职责 | 复用性 |
|--------|------|--------|
| Logo | Logo 展示 | 低 |
| SearchBar | 搜索输入 | 中 |
| ModelSelector | 模型选择下拉 | 低 |
| ThemeToggle | 主题切换按钮 | 高 |
| UserMenu | 用户菜单 | 中 |

**Header 不应包含**:
- 会话列表 (属于 Sidebar)
- 消息相关 (属于 ChatArea)

### 4.2 Sidebar 组件

**拆分内容**:

| 子组件 | 职责 | 复用性 |
|--------|------|--------|
| NewChatButton | 新建会话按钮 | 低 |
| ConversationList | 会话列表容器 | 低 |
| ConversationItem | 单个会话项 | 中 |
| TagList | 标签列表 | 中 |
| TagItem | 单个标签项 | 高 |
| StatsPanel | 数据统计面板 | 低 |
| SidebarFooter | 底部区域 | 低 |

**ConversationItem 进一步拆分**:

| 子组件 | 职责 |
|--------|------|
| ConversationTitle | 会话标题 + 置顶图标 |
| ConversationMeta | 时间 + 消息数 |
| ConversationTags | 标签展示 |
| ConversationActions | 更多操作菜单 |

### 4.3 ChatArea 组件

**拆分内容**:

| 子组件 | 职责 | 复用性 |
|--------|------|--------|
| ChatHeader | 会话信息栏 (可选) | 低 |
| MessageList | 消息列表容器 | 低 |
| MessageItem | 单条消息 | 高 |
| InputArea | 输入区域 | 低 |
| StopButton | 停止生成按钮 | 中 |
| ScrollToBottom | 滚动到底部按钮 | 高 |

**MessageItem 进一步拆分**:

| 子组件 | 职责 |
|--------|------|
| MessageAvatar | 头像 |
| MessageContent | 消息内容 (Markdown) |
| MessageThinking | 思考过程折叠面板 |
| MessageAttachments | 附件列表 |
| MessageActions | 操作按钮 (复制、重新生成、删除) |
| MessageMeta | 元信息 (时间、Token数) |

**MessageContent 进一步拆分**:

| 子组件 | 职责 |
|--------|------|
| MarkdownRenderer | Markdown 渲染 |
| CodeBlock | 代码块 (带复制、高亮) |
| ImagePreview | 图片预览 |

### 4.4 InputArea 组件

**拆分内容**:

| 子组件 | 职责 | 复用性 |
|--------|------|--------|
| AttachmentButton | 附件上传按钮 | 中 |
| ChatInput | 多行文本输入 | 低 |
| SendButton | 发送按钮 | 中 |
| InputHints | 输入提示/建议 | 低 |
| StopGenerationButton | 停止生成按钮 | 中 |

### 4.5 Modal 组件

**拆分内容** (按需加载):

| 组件 | 职责 |
|--------|------|
| ExportModal | 导出会话 |
| TagManagerModal | 标签管理 |
| SettingsModal | 设置页面 |
| ConversationShareModal | 会话分享 (未来) |

## 七、组件通信方式

### 7.1 父子通信

```
父组件 ─────props──────> 子组件
     <────emit事件────
```

**示例**: MessageList -> MessageItem
- props: message, isStreaming
- emit: copy, regenerate, delete

### 7.2 兄弟通信

```
兄弟组件 ─────事件总线/Pinia──────> 兄弟组件
```

**示例**: Sidebar 和 ChatArea
- 通过 Pinia store 共享 currentConversationId

### 7.3 跨层级通信

```
祖先组件 ─────provide/inject──────> 后代组件
```

**示例**: ChatPage -> MessageItem
- provide: handleMessageAction
- inject: sendMessage

### 8.1 路由级懒加载

```javascript
const ChatPage = () => import('@/pages/ChatPage.vue')
```
