# 聊天模块 - 整体布局设计

## 一、页面结构

### 1.1 布局层次

```
┌─────────────────────────────────────────────────────────────────┐
│                        顶部导航栏 (Header)                        │
│  [Logo] [搜索框] [模型选择器] [切换主题] [用户头像]               │
├──────────────┬──────────────────────────────────────────────────┤
│              │                                                   │
│              │           聊天消息区域 (ChatArea)                  │
│   侧边栏     │  ┌─────────────────────────────────────────────┐ │
│  (Sidebar)   │  │                                             │ │
│              │  │            消息列表 (MessageList)            │ │
│  [新建会话]  │  │                                             │ │
│              │  │  ┌───────────────────────────────────────┐  │ │
│  会话列表    │  │  │ 用户消息 / AI回复                       │  │ │
│  - 会话1     │  │  │ (支持 Markdown、代码高亮、思考过程)     │  │ │
│  - 会话2     │  │  └───────────────────────────────────────┘  │ │
│  - 会话3     │  │                                             │ │
│  - ...       │  │  [加载更多提示 / 滚动加载]                   │ │
│              │  └─────────────────────────────────────────────┘ │
│  [标签管理]  │                                                   │
│              │  ┌─────────────────────────────────────────────┐ │
│  数据统计    │  │        输入区域 (InputArea)                 │ │
│  - 今日消息   │  │  [文本输入框] [附件按钮] [发送按钮]          │ │
│  - 总Token   │  │  [停止生成按钮] (生成中显示)                 │ │
│              │  └─────────────────────────────────────────────┘ │
└──────────────┴──────────────────────────────────────────────────┘
```

### 1.2 响应式断点

| 断点 | 屏幕宽度 | 侧边栏 | 布局变化 |
|------|----------|--------|----------|
| 移动端 | < 768px | 隐藏/抽屉式 | 单栏布局，侧边栏通过汉堡菜单展开 |
| 平板 | 768px - 1024px | 可折叠 | 侧边栏可折叠，折叠后只显示图标 |
| 桌面 | > 1024px | 始终显示 | 标准双栏布局 |

## 二、顶部导航栏 (Header)

### 2.1 布局结构

```
┌─────────────────────────────────────────────────────────────────┐
│  🤖 DeepSeek    [🔍 搜索对话...]  [📊 模型选择器 ▼]  [🌙]  👤   │
│  Logo           搜索输入            模型下拉      主题   用户    │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 区域划分

| 区域 | 宽度 | 对齐 | 内容 |
|------|------|------|------|
| Logo区 | 200px | 左对齐 | Logo + 产品名称 |
| 搜索区 | 300px | 居中 | 搜索输入框 (可收缩) |
| 工具区 | 自适应 | 右对齐 | 模型选择、主题切换、用户菜单 |

### 2.3 固定行为

- 始终固定在顶部
- 高度: 56px (移动端) / 64px (桌面端)
- 背景: 白色/深色 (跟随主题)
- 阴影: 轻微底部阴影

---

## 三、侧边栏 (Sidebar)

### 3.1 布局结构

```
┌──────────────────────┐
│   [+ 新建对话]       │  ← 主操作按钮
├──────────────────────┤
│                      │
│  对话列表            │  ← 可滚动区域
│  ┌─────────────────┐ │
│  │ 📌 Python学习   │ │  ← 置顶会话
│  │    今天 10:30   │ │
│  ├─────────────────┤ │
│  │ 💬 日常聊天     │ │
│  │    昨天        │ │
│  │ 🏷️ 工作        │ │  ← 带标签
│  │    3天前       │ │
│  └─────────────────┘ │
│                      │
│  ─────────────────   │  ← 分隔线
│                      │
│  🏷️ 标签 (3)        │  ← 标签管理入口
│  📊 今日: 12消息     │  ← 数据统计
│  💾 总Token: 5.2K   │
│                      │
└──────────────────────┘
```

### 3.2 尺寸规范

| 属性 | 桌面端 | 平板端(折叠) | 移动端 |
|------|--------|--------------|--------|
| 宽度 | 280px | 64px (仅图标) | 100% (抽屉) |
| 最小宽度 | 200px | 64px | - |
| 最大宽度 | 360px | 64px | - |

### 3.3 会话列表项结构

```
┌──────────────────────────────────┐
│ 📌 [图标]  会话标题              │  ← 第一行：图标 + 标题
│           今天 10:30    [⋮]      │  ← 第二行：时间 + 更多操作
│  [标签1] [标签2]                 │  ← 第三行：标签 (可选)
└──────────────────────────────────┘
```

### 3.4 滚动行为

- 会话列表区域独立滚动
- 新消息会话自动滚动到顶部
- 滚动到底部不触发加载（全部在内存）

## 四、聊天主区域 (ChatArea)

### 4.1 布局结构

```
┌─────────────────────────────────────────────────────────────────┐
│                        会话信息栏 (可选)                          │
│  # Python学习讨论         [📌] [🏷️] [⋯] [导出]                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│                        消息列表区域                               │
│  (可滚动、支持无限加载、支持Markdown渲染、代码高亮)              │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                        输入区域                                  │
│  [📎] [文本输入框.....................] [↗️ 发送]               │
│                        [🛑 停止生成] (生成中显示)                │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 会话信息栏 (可选显示)

- 显示当前会话标题
- 操作按钮：置顶、标签、更多、导出
- 可通过设置隐藏

### 4.3 消息列表区域

- 占据剩余所有空间
- 初始滚动到最底部
- 新消息自动滚动到底部
- 用户向上滚动时暂停自动滚动
- 消息间间距: 16px

### 4.4 输入区域

- 固定在底部
- 高度自适应 (最小 56px，最大 200px)
- 支持多行输入
- 生成中显示停止按钮

## 六、尺寸规范总结

| 元素 | 高度 | 备注 |
|------|------|------|
| Header | 56px / 64px | 移动/桌面 |
| Sidebar 宽度 | 280px | 可折叠至 64px |
| 会话列表项 | 64px | 单行 |
| 消息气泡 | 自适应 | 最小高度 40px |
| 输入框 | 56px - 200px | 自适应高度 |
| 底部安全区 | 0px / 34px | 桌面/移动端 |
