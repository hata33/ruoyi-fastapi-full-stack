1. 创建虚拟环境
```
python -m venv .venv
```
2. 激活虚拟环境
```
.venv\Scripts\activate
```
3. 安装依赖
```
pip install -r requirements.txt
```
或
```
uv pip install -e ".[postgres]"
```
4. 运行项目
```
uvicorn app:app --reload
uvicorn app:app --reload --port 9099
```
