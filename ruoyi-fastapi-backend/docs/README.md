# RuoYi-Vue3-FastAPI 后端文档目录说明

本文档提供了 `docs/` 目录下所有文件的组织结构和用途说明，帮助开发者快速定位所需的技术文档。

## 目录概览

```
docs/
├── start/         # 项目启动和核心架构说明
├── public/        # 公共技术和编程规范
├── function/      # 业务功能模块详解
├── feature/       # 框架核心功能特性
├── question/      # 常见技术问题解答
├── bugs/          # 问题修复和注意事项
├── database/      # 数据库相关操作
├── prompt/        # AI 代码生成提示词
└── other/         # 其他参考资料
```

## 推荐阅读顺序

### 新成员入门
1. `start/01-file_structure.md` - 了解项目结构
2. `start/05-数据库连接管理模块.md` - 理解数据库连接
3. `function/01-登录功能.md` - 学习核心功能实现
4. `public/03-Depends.md` - 掌握依赖注入机制
5. `feature/03-Aspect切面功能详解.md` - 理解权限控制

### 进阶开发
1. `feature/04-全局异常处理机制.md` - 异常处理机制
2. `function/08-权限-RBAC-ABAC.md` - 权限模型深入
3. `public/11-数据库锁及并发操作.md` - 并发控制
4. `start/15-链路追踪ASGI中间件.md` - 分布式追踪

### 代码生成
1. `prompt/database2api/01-database_table_generation_prompt.md` - 数据表设计
2. `prompt/database2api/02-vo_model_generation_prompt.md` - VO 模型设计
3. 按顺序阅读 DAO → Service → Controller 提示词
