# Mermaid 流程图

本目录存放项目的 Mermaid 流程图，用于可视化展示项目架构和流程。

## 图表列表

| 文件名 | 描述 |
|--------|------|
| `01-project-overview.md` | 项目概览 - 技术栈和整体架构 |
| `02-login-flow.md` | 登录流程 |
| `03-permission-flow.md` | 权限验证流程 |
| `04-request-lifecycle.md` | 请求生命周期 |
| `05-data-flow.md` | 数据流转流程 |

## 如何使用

这些 Mermaid 图表可以：
1. 在支持 Mermaid 的 Markdown 查看器中直接查看（如 GitHub、VS Code + Mermaid 插件）
2. 在线渲染：https://mermaid.live/
3. 导出为 PNG/SVG 图片

## 查看 VS Code

安装 Mermaid 插件：
- 搜索 "Markdown Preview Mermaid Support"
- 或使用 "Mermaid Preview" 插件

然后按 `Ctrl+K V` 打开预览。
