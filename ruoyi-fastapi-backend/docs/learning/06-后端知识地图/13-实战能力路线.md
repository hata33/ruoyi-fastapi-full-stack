## Level 1: Hello API

### 目标
- 创建第一个 FastAPI 接口
- 理解请求-响应流程
- 掌握基本的路由和参数

### 任务清单
```mermaid
flowchart LR
    Setup[环境搭建] --> Create[创建项目]
    Create --> Route[编写路由]
    Route --> Param[处理参数]
    Param --> Response[返回响应]

    ```

### 具体任务
- [ ] 安装 Python 和 FastAPI
- [ ] 创建第一个接口 `GET /hello`
- [ ] 添加路径参数 `GET /hello/{name}`
- [ ] 添加查询参数 `GET /hello?name=World`
- [ ] 返回 JSON 响应
- [ ] 使用 Swagger 文档测试

### 关键技能
- FastAPI 基础
- 路由定义
- 参数处理
- JSON 响应

### 预计时间
1 周

---

## Level 2: CRUD 应用

### 目标
- 实现完整的 CRUD 操作
- 理解数据模型
- 使用数据库持久化

### 任务清单
```mermaid
flowchart TB
    Design[设计数据模型] --> Create[创建接口]
    Create --> Read[查询接口]
    Read --> Update[更新接口]
    Update --> Delete[删除接口]

    ```

### 具体任务
- [ ] 设计用户表结构
- [ ] 定义 SQLAlchemy 模型
- [ ] 实现 `POST /users` 创建用户
- [ ] 实现 `GET /users` 获取用户列表
- [ ] 实现 `GET /users/{id}` 获取单个用户
- [ ] 实现 `PUT /users/{id}` 更新用户
- [ ] 实现 `DELETE /users/{id}` 删除用户
- [ ] 添加参数验证
- [ ] 统一响应格式

### 关键技能
- SQLAlchemy ORM
- 数据模型设计
- CRUD 操作
- Pydantic 验证

### 预计时间
2 周

---

## Level 3: 认证授权

### 目标
- 实现用户认证
- 使用 JWT Token
- 基础权限控制

### 任务清单
```mermaid
flowchart TB
    Login[登录接口] --> JWT[生成JWT]
    JWT --> Verify[验证Token]
    Verify --> Protect[保护接口]
    Protect --> Permission[权限检查]

    ```

### 具体任务
- [ ] 实现密码哈希存储 (bcrypt)
- [ ] 创建 `POST /auth/login` 登录接口
- [ ] 生成 JWT Token
- [ ] 实现 Token 验证依赖
- [ ] 保护需要登录的接口
- [ ] 实现基础的 RBAC 权限
- [ ] 添加刷新 Token 机制

### 关键技能
- 密码哈希
- JWT 认证
- 依赖注入
- 权限控制

### 预计时间
2 周

## 学习建议

## 学习原则

### 循序渐进
- 不要跳级，每个 Level 都要扎实掌握
- 完成当前 Level 的所有任务再进入下一级
- 遇到困难时回顾基础知识

### 实践为主
- 理论结合实践
- 每个知识点都要动手实现
- 记录遇到的问题和解决方案

### 持续改进
- 定期重构代码
- 应用新学到的知识优化旧代码
- 学习优秀的开源项目

### 寻求反馈
- 代码审查
- 技术分享
- 参与开源社区
