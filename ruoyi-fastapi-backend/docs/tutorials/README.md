> 以记账系统开发为主线，从零基础到独立搭建后端项目

## 🎯 学习目标

通过完成一个完整的记账系统，掌握：
- Python 后端开发必备技能
- FastAPI 框架的实践应用
- 数据库设计与 ORM 操作
- 后端开发的核心思维方式

## 💡 记账系统功能

### 核心功能
- 用户注册登录
- 记账记录 CRUD
- 账户管理
- 分类管理
- 统计报表

### 技术栈
- **框架**: FastAPI
- **数据库**: MySQL + SQLAlchemy
- **认证**: JWT
- **缓存**: Redis
- **部署**: Docker

## 📖 学习建议

### 学习方法
1. **跟着做**: 每章代码都要亲手敲一遍
2. **多思考**: 理解"为什么"比"怎么做"更重要
3. **做笔记**: 建立自己的知识体系
4. **敢重构**: 后期学到新思维，回头看代码

### 检查点
每章末尾都有检查点，确保理解后再继续：
- ✅ 能解释核心概念
- ✅ 能独立完成练习
- ✅ 能应用到新场景

## 🎓 学习前提

- Python 基础了解（变量、循环、条件判断）
- 了解 HTTP 是什么（不用深入）
- 有动手实践的意愿

**不需要**:
- ❌ 不需要深厚的 Python 功底
- ❌ 不需要数据库经验
- ❌ 不需要前端知识

## 📞 遇到问题？

- 仔细阅读代码注释
- 查看相关章节的"为什么"部分
- 对比自己的代码和示例代码
- 在项目仓库提 Issue
